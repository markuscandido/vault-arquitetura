# Vault Arquitetura

Este repositório é uma base de conhecimento persistente e estruturada (LLM Wiki) focada em padrões de arquitetura de software, baseada principalmente na obra de Martin Fowler, *Patterns of Enterprise Application Architecture* (PEAA).

## Estrutura do Projeto

- `raw/`: Documentos de origem imutáveis (capítulos, artigos, notas).
- `wiki/`: Base de conhecimento gerada e mantida por IA.
    - `index.md`: Catálogo central de todo o conteúdo.
    - `log.md`: Registro cronológico de operações (ingestões, consultas, manutenção).
    - `concepts/`: Definições detalhadas de padrões e conceitos arquiteturais.
    - `sources/`: Resumos e metadados das fontes consultadas.

## Diretrizes de Idioma

Todo o conteúdo da wiki é mantido em **Português Brasileiro (PT-BR)** para garantir clareza e acessibilidade, respeitando a terminologia técnica padrão da indústria em inglês quando necessário.

## Como Usar

Este repositório foi projetado para ser utilizado com o **Gemini CLI** ou ferramentas de visualização de Markdown como **Obsidian**.

- Para navegar, comece pelo `wiki/index.md`.
- Para acompanhar a evolução do conhecimento, consulte `wiki/log.md`.
